# Gas Town Building Sprites

Mad Max-themed building sprites for the Gas Town visual dashboard.

## Sprites

### Mayor (mayor.svg)
- **Size:** 120x160 (largest)
- **Style:** Citadel/fortress tower with battlements
- **Features:** Multi-story tower, flag on top, scaffolding details
- **Role:** Command center, highest authority

### Refinery (refinery.svg)
- **Size:** 100x100 (medium)
- **Style:** Industrial complex with pipes and tanks
- **Features:** Multiple storage tanks, smokestacks, piping network
- **Role:** Processing center, merge queue handler

### Witness (witness.svg)
- **Size:** 80x120 (medium)
- **Style:** Observation tower with lattice structure
- **Features:** Lookout cabin, searchlight, warning signals
- **Role:** Monitoring and health checks

### Deacon (deacon.svg)
- **Size:** 90x110 (medium)
- **Style:** Messenger station with communications equipment
- **Features:** Antenna mast, satellite dish, signal equipment
- **Role:** Message routing and coordination

## Design Notes

All sprites use a Mad Max post-apocalyptic aesthetic:
- Rusty, weathered metal tones (browns, grays, oranges)
- Industrial, ramshackle construction
- Functional but rough appearance
- Dark color palette with rust accents

## Usage

These SVG files can be:
- Embedded directly in HTML
- Loaded as separate assets via `<img>` tags
- Styled with CSS for hover effects or state changes
- Scaled without quality loss (vector format)
